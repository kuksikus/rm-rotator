Demo page of rm-rotator

Example #1

Standard use
$(function() {
	$('#example1').Rotator({
		count: 60,
		start: 0,
		prefix: 'demo_images/violet/img_0_0_',
		postfix: '.jpg',
		height: 300
	});
});

<div id="example1">
	<img src="demo_images/violet/img_0_0_0.jpg">
</div>

Example #2

Auto rotate and controls
$(function() {
	$('#example2').Rotator({
		count: 60,
		start: 0,
		prefix: 'demo_images/violet/img_0_0_',
		postfix: '.jpg',
		height: 300,
		auto_rotate: true,
		controls: ['rotate']
	});
});

<div id="example2">
	<img src="demo_images/violet/img_0_0_0.jpg">
</div>

All options

Option Description Required Default value Values Example
count (int) Count of images yes count: 60
prefix (string) Path to images and image prefix yes prefix: 'path/to/images/image_'
postfix (string) Image postfix yes postfix: '.jpg'
start (int) Set start frame 0 start: 15
add_zeros (bool) Set 'true' if filenames like image09 image10 false add_zeros: true
width\height (int) Set image sizes original size height: 300
zoom_max (float) Set max zoom 1.5 zoom_max: 2.3
zoom_step (float) Set zoom step 0.2 zoom_step: 0.5
zoom (bool) Enable or disable zoom true zoom: false
rotate_delay (int) in milliseconds Rotate delay for speed control 40 (ms) rotate_delay: 100
auto_rotate (bool) Enable or disable auto rotate false auto_rotate: true
controls (array) List of enabled controls ['rotate', 'all'] controls: ['rotate']
